# Kinetic Games Responsive Landing Page

This project is a responsive landing page for Kinetic Games. It was built as a simple front-end site that presents the brand in a clean, modern way and works well on both desktop and mobile screens.

## What it includes

- A full landing page layout
- Responsive design for different screen sizes
- A hero section, navigation, and visual content
- Assets stored in the `public` folder

## What I did

I created the landing page structure and styled it so it feels polished and easy to use. The page is meant to give a strong first impression and show the Kinetic Games brand clearly.

## Live Demo

You can view the deployed version here:

https://kinetic-games.vercel.app

## How to run it

Install dependencies:

```bash
pnpm install
```

Start the project:

```bash
pnpm start
```

The site will run locally on port 3000.
